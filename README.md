# Student Management System

## Description
A console-based Student Management System developed in Python to manage student records using file handling.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Data Storage using File Handling

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
python student_management.py
